# Emirates Airlines Landing  Clone Page

A responsive frontend recreation of the Emirates Airlines website landing page. Built as part of the User Interface Design (UID) course assignment.

## ✨ Live Demo

View Live → [https://github.com/GandrathiSubramanyam62-rgb/gandrathisubramanyam]

## ✨ Features

* Responsive design for Mobile, Tablet, and Desktop
* Modern UI inspired by Emirates Airlines
* Smooth animations and transitions
* Interactive navigation bar
* Attractive flight booking interface
